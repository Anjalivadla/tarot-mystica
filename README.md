Tarot Mystica – Interactive Tarot Reading Website

About
Tarot Mystica is a simple website made using HTML and CSS. It gives a basic tarot reading experience with different pages for Past, Present, and Future.

This project was created to practice frontend development and understand how multiple web pages are connected.

Features
- Separate pages for Past, Present, and Future readings  
- Simple navigation between pages  
- Basic UI using HTML and CSS  
- Clean and easy layout  

Technologies Used
- HTML  
- CSS  

Pages
- index.html – Home page  
- past.html – Past reading page  
- present.html – Present reading page  
- future.html – Future reading page  

What I Learned
- How to create multiple HTML pages  
- How to link pages together  
- Basics of CSS styling  
- How to organize a small web project  

Note
This is a frontend-only project. No backend or database is used.
